title: Advanced Transcription Model Management
description: A Rich User Experience
---

For users who rely on the on-device transcription and translation features, **aj** provides an advanced model management screen with a rich user experience.

**A State-Aware UI for a Seamless Experience:**

The model management screen is designed to be fully state-aware, providing you with instant feedback on the status of each model.

*   **Clear Status Indicators:** Each model in the list is clearly labeled with its current status: **Active**, available, or downloading.
*   **Real-Time Download Progress:** When you download a new model, the UI displays a `LinearProgressIndicator`.
*   **A Clear Workflow:** The interface is designed to guide you through the model management workflow.
